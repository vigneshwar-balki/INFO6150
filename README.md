
# INFO6150 Assignment 3

## Assignment Summary
This project implements a dynamic student awards table for INFO6150. The goal is to allow users to add, edit, delete, and manage student award entries interactively in the browser.

## Implementation Steps
1. **Basic Structure:**
	- Set up the initial HTML structure with a table skeleton and control buttons.
2. **Initial Data:**
	- Populated the table with three sample student rows and set up hidden detail rows for expansion.
3. **Dynamic Addition:**
	- Implemented the ability to add new students dynamically, ensuring sequential ordering and user alerts.
4. **Font Awesome & Icons:**
	- Integrated Font Awesome for action icons (edit, delete, etc.).
5. **Checkbox Selection & Highlighting:**
	- Added checkboxes for row selection, row highlighting, and dynamic column management.
6. **Submit Button State:**
	- Enabled/disabled the submit button based on selection state, with supporting CSS updates.
7. **Row Deletion:**
	- Enabled row deletion with success alerts and automatic table reordering.
8. **Edit & Row Expansion:**
	- Added edit prompts and row expansion/collapse toggles for detailed student info.
9. **HTML/CSS Refactor:**
	- Moved the Add Student HTML snippet to the main HTML file and updated CSS for improved UI.
